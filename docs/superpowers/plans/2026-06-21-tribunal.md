# Tribunal Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a working Casper Testnet prototype of Tribunal, a multi-agent RWA verification oracle where specialist GenLayer judges each verify one facet of a claim and an Odra contract federates their verdicts into one auditable on-chain answer with staked reputation.

**Architecture:** Off-chain GenLayer judges (one per facet, each its own consensus) post verdicts onto a Casper Odra contract signed by their own keys. The contract federates verdicts with a critical-facet veto plus reputation-weighted aggregation, stores the canonical verdict, and stakes/slashes reputation on resolution. An off-chain TypeScript orchestrator dispatches claims and relays verdicts; a thin Next.js UI demos it.

**Tech Stack:** Odra 2.8 (Rust/WASM) on Casper Testnet; GenLayer intelligent contracts (Python) for judges; TypeScript orchestrator using genlayer-js + casper-js-sdk; Next.js + CSPR.click for the UI; x402 for paid evidence endpoints.

**Sequencing rule:** Milestones are ordered so the proof-of-reserves vertical slice reaches Casper Testnet at M3. M0 to M3 is the qualifying entry. M4 to M6 strengthen it. If time slips, we still qualify on M3.

```rust
pub fn federate(inputs: &[FacetInput], veto_threshold: Bps, backed_threshold: Bps, notbacked_threshold: Bps) -> Federated {
    // Pass 1: critical veto. Any critical FAIL whose reputation-weighted confidence
    // clears the veto threshold sinks the whole claim.
    for i in inputs {
        if i.config.critical && i.vote == Vote::Fail {
            let weighted = (i.confidence as u64 * i.reputation as u64 / 10_000) as Bps;
            if weighted >= veto_threshold {
                return Federated { status: ClaimStatus::NotBacked, score: 0 };
            }
        }
    }

    // Pass 2: reputation- and weight-weighted aggregate. PASS=+conf, FAIL=-conf, UNCERTAIN=0.
    // Numerator can go negative, so work in i128 then clamp to [0, 10000].
    let mut num: i128 = 0;
    let mut den: i128 = 0;
    for i in inputs {
        let w = (i.config.weight as i128) * (i.reputation as i128);
        let signed_conf: i128 = match i.vote {
            Vote::Pass => i.confidence as i128,
            Vote::Fail => -(i.confidence as i128),
            Vote::Uncertain => 0,
        };
        num += w * signed_conf;
        den += w;
    }
    let score_raw = if den == 0 { 0 } else { num / den };
    let score = score_raw.clamp(0, 10_000) as Bps;

    let status = if score >= backed_threshold {
        ClaimStatus::Backed
    } else if score <= notbacked_threshold {
        ClaimStatus::NotBacked
    } else {
        ClaimStatus::Disputed
    };
    Federated { status, score }
}
```

```rust
pub fn update_reputation(prior: Bps, correct: bool, step: Bps, floor: Bps) -> Bps {
    if correct {

    } else {
```

```rust
use odra::prelude::*;
use odra::{Var, Mapping, List};
use crate::types::*;
use crate::federation::{federate, FacetInput};
use crate::reputation::update_reputation;

#[odra::module]
pub struct Tribunal {
    admin: Var<Address>,
    // facet config keyed by facet_id
    facets: Mapping<u8, FacetConfig>,
    facet_ids: List<u8>,
    // judge reputation in bps, keyed by judge address
    reputation: Mapping<Address, Bps>,
    registered: Mapping<Address, bool>,
    // claim_id -> status / score / resolved
    claim_status: Mapping<u64, ClaimStatus>,
    claim_score: Mapping<u64, Bps>,
    next_claim_id: Var<u64>,
    // (claim_id, facet_id) -> submitted verdict
    verdicts: Mapping<(u64, u8), SubmittedVerdict>,
    verdict_present: Mapping<(u64, u8), bool>,
    // tunables (bps)
    veto_threshold: Var<Bps>,
    backed_threshold: Var<Bps>,
    notbacked_threshold: Var<Bps>,
    rep_step: Var<Bps>,
    rep_floor: Var<Bps>,
    starting_rep: Var<Bps>,
}

#[odra::module]
impl Tribunal {
    pub fn init(&mut self) {
        self.admin.set(self.env().caller());
        self.next_claim_id.set(0);
        self.veto_threshold.set(6000);
        self.backed_threshold.set(7000);
        self.notbacked_threshold.set(4000);
        self.rep_step.set(500);
        self.rep_floor.set(1000);
        self.starting_rep.set(5000);
    }

    pub fn configure_facet(&mut self, facet_id: u8, weight: u32, critical: bool) {
        self.assert_admin();
        if self.facets.get(&facet_id).is_none() {
            self.facet_ids.push(facet_id);
        }
        self.facets.set(&facet_id, FacetConfig { facet_id, weight, critical });
    }

    pub fn register_judge(&mut self, judge: Address) {
        self.assert_admin();
        if !self.registered.get(&judge).unwrap_or(false) {
            self.registered.set(&judge, true);
            self.reputation.set(&judge, self.starting_rep.get_or_default());
        }
    }

    pub fn open_claim(&mut self) -> u64 {
        let id = self.next_claim_id.get_or_default();
        self.next_claim_id.set(id + 1);
        self.claim_status.set(&id, ClaimStatus::Open);
        id
    }

    pub fn submit_verdict(&mut self, claim_id: u64, facet_id: u8, vote: Vote, confidence: Bps, genlayer_proof: String) {
        let judge = self.env().caller();
        if !self.registered.get(&judge).unwrap_or(false) {
            self.env().revert(Error::NotRegistered);
        }
        if self.verdict_present.get(&(claim_id, facet_id)).unwrap_or(false) {
            self.env().revert(Error::DuplicateVerdict);
        }
        self.verdicts.set(&(claim_id, facet_id), SubmittedVerdict { facet_id, judge, vote, confidence, genlayer_proof });
        self.verdict_present.set(&(claim_id, facet_id), true);
    }

    pub fn finalize(&mut self, claim_id: u64) -> ClaimStatus {
        let mut inputs: Vec<FacetInput> = Vec::new();
        let n = self.facet_ids.len();
        for idx in 0..n {
            let fid = self.facet_ids.get(idx).unwrap();
            if self.verdict_present.get(&(claim_id, fid)).unwrap_or(false) {
                let v = self.verdicts.get(&(claim_id, fid)).unwrap();
                let rep = self.reputation.get(&v.judge).unwrap_or(0);
                let cfg = self.facets.get(&fid).unwrap();
                inputs.push(FacetInput { config: cfg, vote: v.vote, confidence: v.confidence, reputation: rep });
            }
        }
        let out = federate(&inputs, self.veto_threshold.get_or_default(), self.backed_threshold.get_or_default(), self.notbacked_threshold.get_or_default());
        self.claim_status.set(&claim_id, out.status.clone());
        self.claim_score.set(&claim_id, out.score);
        out.status
    }

    // ground_truth: the facets (by id) that were actually true. Used to score judges.
    pub fn resolve_claim(&mut self, claim_id: u64, truth_pass_facets: Vec<u8>) {
        self.assert_admin();
        let n = self.facet_ids.len();
        let step = self.rep_step.get_or_default();
        let floor = self.rep_floor.get_or_default();
        for idx in 0..n {
            let fid = self.facet_ids.get(idx).unwrap();
            if self.verdict_present.get(&(claim_id, fid)).unwrap_or(false) {
                let v = self.verdicts.get(&(claim_id, fid)).unwrap();
                let truth_pass = truth_pass_facets.contains(&fid);
                let judge_pass = v.vote == Vote::Pass;
                let correct = truth_pass == judge_pass; // Uncertain counts as wrong by construction
                let prior = self.reputation.get(&v.judge).unwrap_or(0);
                self.reputation.set(&v.judge, update_reputation(prior, correct, step, floor));
            }
        }
    }

    // --- views ---
    pub fn get_status(&self, claim_id: u64) -> ClaimStatus { self.claim_status.get(&claim_id).unwrap_or(ClaimStatus::Open) }
    pub fn get_score(&self, claim_id: u64) -> Bps { self.claim_score.get(&claim_id).unwrap_or(0) }
    pub fn get_reputation(&self, judge: Address) -> Bps { self.reputation.get(&judge).unwrap_or(0) }
    // per-facet breakdown so a consumer can drill into a finalized verdict
    pub fn get_verdict(&self, claim_id: u64, facet_id: u8) -> Option<SubmittedVerdict> { self.verdicts.get(&(claim_id, facet_id)) }

    fn assert_admin(&self) {
        if self.env().caller() != self.admin.get().unwrap() {
            self.env().revert(Error::NotAdmin);
        }
    }
}

#[odra::odra_error]
pub enum Error {
    NotAdmin = 1,
    NotRegistered = 2,
    DuplicateVerdict = 3,
}
```
